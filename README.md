# Python AI Project - Ain Shams University

This repository contains the AI project for the Ain Shams University course.

## Project Structure
- `data/`: Directory for storing datasets.
- `models/`: Directory for saved trained models (ignored by git).
- `notebooks/`: For Jupyter notebooks used in data exploration and model training.
- `src/`: Source code for the project.
